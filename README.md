# My Vim Configuration

This is my vim-config versioned with thanks to romainl 
for his [post](https://stackoverflow.com/a/18203545).


## Installation

		$ cd ~
		$ git clone https://github.com/hennio/vimconfig.git .vim
		$ echo "runtime vimrc" > .vimrc
